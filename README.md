# 🚀 HOW TO MAKE YOUR WEBSITE LIVE

You have **three easy options**. Pick the one that feels easiest.

## OPTION 1: GitHub Pages (Recommended — FREE, Permanent URL)

### Step 1 — Create a GitHub Account
1. Go to **https://github.com**
2. Click **Sign Up** and create a free account (use your email)

### Step 2 — Create a New Repository
1. Click the **"+"** icon in the top-right corner → **"New repository"**
2. Name it: **`ishyna-luxe`**
3. Set visibility to **Public**
4. ✅ Check "Add a README file"
5. Click **Create repository**

### Step 3 — Upload Your Website Files
1. On your new repo page, click **"Add file"** → **"Upload files"**
2. Drag and drop **EVERYTHING** from the `ishyna-luxe` folder:
   - `index.html`
   - The entire `assets/` folder (with images and videos inside)
3. Scroll down, add commit message: **"Initial website upload"**
4. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repository's **Settings** tab (top menu)
2. Click **Pages** on the left sidebar
3. Under "Source", select **Deploy from a branch**
4. Branch: **main** · Folder: **/ (root)**
5. Click **Save**
6. Wait 1–2 minutes. You'll see a green box: ✅ "Your site is live at..."

### Step 5 — Your Live URL
Your website will be available at:

```
https://YOUR-USERNAME.github.io/ishyna-luxe/
```

Share this link in your assignment! 🎉

---

## OPTION 2: Netlify Drop (Fastest — 30 Seconds, No Account Required)

1. Go to **https://app.netlify.com/drop**
2. Drag and drop the **entire `ishyna-luxe` folder** onto the page
3. Wait 10 seconds...
4. You'll get an instant URL like: `https://amazing-name-12345.netlify.app`
5. (Optional) Create a free Netlify account to rename the URL to something like `ishynaluxe.netlify.app`

**Best for:** Instant demo, no coding experience needed.

---

## OPTION 3: Vercel (Professional Option)

1. Go to **https://vercel.com** and sign up with GitHub
2. Click **"Add New Project"**
3. Import your `ishyna-luxe` GitHub repo
4. Click **Deploy**
5. Live in 30 seconds at `https://ishyna-luxe.vercel.app`
